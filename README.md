## 📋 섹션 구성

| 섹션 | 내용 |
|------|------|
| **Hero** | 이름, 직함, 소개, CTA 버튼 |
| **About** | 상세 소개 및 프로필 정보 카드 |
| **Skills** | SVG 원형 진행바 기술 스택 |
| **Projects** | 타임라인 스타일 포트폴리오 3개 |
| **Contact** | 연락처 + 문의 폼 |

---

## 🛠️ 기술 스택

- **HTML5** — 시맨틱 마크업
- **CSS3** — 반응형 레이아웃, 애니메이션
- **JavaScript (ES6+)** — DOM 조작, IntersectionObserver
- **Tailwind CSS** — CDN 방식 유틸리티 스타일링
- **Google Fonts** — Noto Sans KR

---

## ✨ 주요 기능

- 📱 **완전 반응형** — 모바일 / 태블릿 / 데스크탑 최적화
- 🍔 **모바일 햄버거 메뉴** — 소형 화면 네비게이션
- 🎞️ **스크롤 페이드인 애니메이션** — IntersectionObserver 활용
- 🔵 **SVG 원형 스킬 차트** — stroke-dashoffset 애니메이션
- 📩 **연락처 폼** — 클라이언트 사이드 제출 처리

## 📁 파일 구조

```
my-profile-site/
├── index.html        ← 메인 포트폴리오 사이트 (단일 파일)

├── .nojekyll         ← GitHub Pages Jekyll 비활성화
├── .gitignore        ← Git 제외 파일 설정
└── README.md         ← 프로젝트 소개 (현재 파일)
```
